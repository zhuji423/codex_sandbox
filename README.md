# 本地贪吃蛇游戏

这个目录下已经创建好一个**可离线运行**的贪吃蛇小游戏：`snake_game.html`。

## 直接运行（推荐）

1. 在文件管理器里双击 `snake_game.html`；
2. 或者在浏览器中直接打开该文件（`file://.../snake_game.html`）。

> 不需要安装依赖，不需要启动后端服务。

## 可选：用本地静态服务运行

如果你更习惯通过 `http://localhost` 访问：

```bash
python3 -m http.server 8000
```

然后打开：<http://localhost:8000/snake_game.html>

## 操作说明

- 移动：方向键 或 `WASD`
- 暂停/继续：空格
- 开始/重开：点击「开始 / 重新开始」
- 速度切换：点击「切换速度」
